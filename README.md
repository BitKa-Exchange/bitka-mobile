# bitka

A bitkub app replica for educational purpose.

## Install Dependencies
```bash
flutter pub get
```

## How to Run
- ### Development
```bash
flutter run
```
- ### Production
```bash
flutter run --dart-define=ENVIRONMENT=.env.production --release
```
